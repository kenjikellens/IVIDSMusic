# Code Style and Layout Rules

- Prefer using percentage `%` or viewport units (`vw`, `vh`) instead of fixed pixel `px` values for container sizing (e.g., specifying what portion of the viewport it fills).
- Maintain, respect, and design within the 5 responsive width breakpoints (1200px, 1024px, 768px, 700px, and 600px).
